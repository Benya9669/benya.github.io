TTL+ Kernel by benya9669

Hello everyone, the goal of this kernel is to be stable but guarantee many tunable settings to all advanced users

Features

  • Compiled with the latest Proton Clang
  • Updated with the latest sources
  • Cpu Governors: blu_active, all stocks
  • CPU underclock to 230Mhz
  • GPU undeclock to 180Mhz
  • TCP congestion algorithms: westwood (tunable), all stocks
  • Wireguard support
  • Boeffla_wl_blocker (generic wakelock blocker driver)
  • TTL and HL target support

Installation

  1. Reboot in Recovery
  2. Wipe Dalvik, Cache (Recommended)
  3. Flash kernel
  4. Boot
